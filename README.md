
# Portfolio Stress Testing & Scenario Analysis in Python

## Overview

This project implements a **historical scenario-based stress testing framework** for an equity portfolio using Python and real market data.

The objective is to evaluate **portfolio losses under major financial crises** and demonstrate practical applications of **market risk analytics aligned with FRM concepts**.

## Stress Scenarios Covered

* Global Financial Crisis (2008)
* China Market Slowdown (2015â€“2016)
* COVID-19 Market Crash (2020)
* Global Rate-Hike Cycle (2022)

## Methodology

1. Download historical equity prices using `yfinance`
2. Compute daily returns for each security
3. Apply historical crisis-period shocks to current portfolio prices
4. Estimate **portfolio-level stressed P&L**
5. Visualize scenario losses using Python charts
